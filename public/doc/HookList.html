Methods

call(val) → {*}

Call each function entry in hook list, bound to this.owner.

The first function is called with the arguments as given to this method. When a called function returns a value besides undefined, val is set to that value for the next function. The final state of val is returned at the end of the iteration.

Thus, if constituent functions return a value, this operation serves to "filter" a value from one function to the other. Conversely, if constituent functions do not return an explicit value, this method essentially iterates over these functions with identical arguments.

The former mechanism is used by Model#filters, while the latter is used by CanvasAdapter#onDrawSelector, and, when drawing individual cells, by CanvasAdapter#onDrawCell.

callParallel(array)

Call each function entry for every value in the given array, completing each function for all elements in the array before moving on to the next.

Used by CanvasAdapter#draw to finish each successive drawing function for all cells in turn, allowing more complex intercellular drawings.
