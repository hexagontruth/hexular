Class: OffsetModel

OffsetModel()

Class representing an offset, i.e. rectangular, topology.

In an offset topology, cells describe a cols * rows grid where every other row is staggered one apothem-length along the x axis. This is useful when trying to fit a grid into a rectangular box, but may cause undesirable wraparound effects. (These effects may be mitigated by using Hexular.filters.edgeFilter.)
