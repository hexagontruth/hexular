(static) ruleBuilder(ruleDef, opts) → {function}

Generates an elementary rule based on the state of a cell's neighbors plus optionally itself.

The most significant (left-most) bit represents the lowest neighbor number in the selected range, while the least significant bit represents the highest. Thus, the same rule masks can be used with opts.range set to either [0, 7] or [1, 7] (default).

Modeled roughly after Wolfram's Elementary Cellular Automaton rules.
